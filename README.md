# 📚 Easy Book
A Flutter application for exploring and previewing free books online.

## 🌟 Features
- 🏗️ MVVM Architecture
- 🔍 Book Search 
- 📖 Book Details
- 👀 Content Preview
- 🌐 Google Books API Integration

## 🚀 Getting Started
1. Clone the repository
2. Run `flutter pub get`
3. Connect an emulator or device
4. Run `flutter run`

## 💻 Technologies
- 📱 Flutter
- 🏛️ MVVM Architecture
- 🔗 Google Books API
